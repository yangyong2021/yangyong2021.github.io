LC42接雨水

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

示例 2：

提示：

  • n == height.length
  • 1 <= n <= 2 * 104
  • 0 <= height[i] <= 105

解法：找到最左边最高的，和最右边最高的，能接住雨水的由最低的决定，则min(r_max, l_max) - 当前位置的高度，就是所接的雨水
